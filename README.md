# BDH House
Site statique avec formulaire multi-étapes pour candidatures.

## Déploiement
1. Crée un compte sur [Formspree](https://formspree.io/) et récupère ton Form ID.
2. Remplace `yourFormID` dans `index.html` par ton Form ID.
3. Pousse le projet sur GitHub.
4. Connecte le dépôt à [Vercel](https://vercel.com/) et déploie.
5. Le site est prêt à être utilisé !
